About

This is a simple web application that has pulled together several tools. The application is written in Python upon a Flask framework. It is drawing data from a data loaded in via CSV. It is implementing a simple Bootstrap template, alterations have been made to the HTML, CSS, and JavaScript. It is also using Heroku as a it's platform as a service.
